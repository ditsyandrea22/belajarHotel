# Belajar Hotel

Repositori ini berisi proyek pembelajaran untuk sistem manajemen hotel sederhana.

## Fitur

- Manajemen Kamar
  - Tambah kamar
  - Lihat daftar kamar
  - Update status kamar
  - Hapus kamar

- Manajemen Tamu
  - Check-in tamu
  - Check-out tamu
  - Lihat daftar tamu

- Laporan
  - Laporan pendapatan
  - Laporan okupansi kamar

## Instalasi

🚀 Cara Menggunakan
