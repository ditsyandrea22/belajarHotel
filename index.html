Tentang Proyek

Ini adalah proyek belajar untuk membuat sistem manajemen hotel sederhana. Proyek ini dikembangkan sebagai bahan pembelajaran untuk memahami konsep pemrograman web fullstack dan manajemen basis data.

Teknologi Modern

Menggunakan teknologi terbaru dalam pengembangan web untuk performa optimal

Responsive Design

Tampilan yang optimal di semua perangkat, dari desktop hingga mobile

Database Terstruktur

Struktur database yang efisien untuk manajemen data hotel

Fitur Utama

Manajemen Kamar

Sistem lengkap untuk mengelola ketersediaan kamar, tipe kamar, dan harga

Pemesanan Online

Proses booking yang mudah dan cepat melalui website

Laporan Pendapatan

Analisis pendapatan dan statistik pemesanan

Antarmuka Administrasi

Dashboard admin untuk mengelola seluruh sistem

Dokumentasi

Cara Menggunakan

  1. Clone atau download repository
  2. Import database dari file SQL yang tersedia
  3. Konfigurasi koneksi database di file config.php
  4. Jalankan di server web lokal (XAMPP, WAMP, dll.)
  5. Akses melalui browser
