Tentang Proyek

Ini adalah proyek belajar untuk membuat sistem manajemen hotel sederhana. Proyek ini dikembangkan sebagai bahan pembelajaran untuk memahami konsep pemrograman web dan manajemen basis data.

Fitur Utama

  • Manajemen Kamar
  • Pemesanan Online
  • Laporan Pendapatan
  • Antarmuka Administrasi

Teknologi yang Digunakan

HTML, CSS, JavaScript, PHP, MySQL

Lihat di GitHub Download Proyek

Cara Menggunakan

  1. Clone atau download repository
  2. Import database dari file SQL yang tersedia
  3. Konfigurasi koneksi database di file config.php
  4. Jalankan di server web lokal (XAMPP, WAMP, dll.)
  5. Akses melalui browser
